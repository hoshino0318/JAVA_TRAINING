# ex12_02 answer.txt

・PassengerVehicle オブジェクトの定員を負の値に設定しようとした
  引数が不正なことを例外として通知する．

・オブジェクトがその初期値を設定するのに使用する設定ファイルに，文法エラーが見つかった
  文法エラーを例外として通知する．

・プログラマが指定した単語を文字列の配列から検索するメソッドが，その単語を発見できない
  -1 や null など発見できなかったことを表す返り値を用意する．

・open メソッドへ指定されたファイルが存在しない．
  No such file or directory のような通知を例外として投げる

・open メソッドへ指定されたファイルは存在するが，セキュリティにより使用できない
  権限がないことを例外として投げる．

・リモートのサーバプロセスにネットワークコネクションを確立しようとするが，リモートのマシンと接続できない
  タイムアウトを通知する (例外ではない)

・リモートのサーバプロセスとのやり取りの最中に，ネットワークコネクションが切れる．
  ネットワークが切れたことを通知する (例外ではない)
